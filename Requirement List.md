## Requirement List

| No. | Requirement List | Use case(s) |
|---|---|---|
|  | 시스템 이용을 위해 회원가입을 해야한다. 필수 입력 정보는 ID,비밀번호,이름,전화번호이다.  | 회원가입  |
| | ID와 비밀번호로 로그인 한다. 고정 ID와 비밀번호로 로그인한다. | 로그인 |
| | 언제든 로그아웃할 수 있다. | 로그아웃 |
| | 언제든 탈퇴할 수 있다. 탈퇴 시 해당 회원의 모든 이용 권한과 데이터가 삭제된다. | 회원탈퇴 |
|  | 새로운 설문을 등록한다. 이때 입력 정보는 설문 제목, 설문 설명, 설문 문항, 응답 항목, 설문 시작 시각, 설문 마감 시각이다.   | 설문 등록  |
| | 등록된 모든 설문 리스트를 조회한다. | 설문 리스트 조회 |
| | 특정 설문 항목을 선택하면 상세 내용을 팝업 창으로 띄운다. | 설문 상세 조회 |
| | 특정 설문 항목을 선택하여 설문을 삭제한다. | 설문 삭제 |
| | 응답한 모든 설문을 한 화면에서 조회한다. 이때 출력 정보는 설문 제목, 응답 내용, 응답 일시이다. | 응답한 설문 조회 |
| | 현재 진행 중인 설문에 한해 제출한 응답을 수정한다. 수정 화면에는 가장 최근 수정 날짜가 표시되며, 수정 버튼을 눌러 완료한다. | 응답 수정 |
| | 현재 진행 중인 설문에 한해 제출한 응답을 취소한다. | 응답 취소 |
| | 관리자가 설문 응답 통계 정보를 조회한다. 검색 기간은 최근 1주일, 1개월, 1년 중에서 선택할 수 있으며, 출력 정보는 전체 설문 수, 종료된 설문 수, 총 응답 수이다. | 통계 조회 |
| | 현재/예정/종료된 설문 리스트 상태를 선택하고 키워드를 입력하여 조건에 맞는 설문 리스트를 검색하고 출력할 수 있다. | 설문 검색 |
| | 검색된 리스트에서 특정 설문을 선택하여 제목, 설명, 문항, 응답 항목 등의 상세 정보를 조회할 수 있으며, 현재 진행 중인 설문에 한하여 응답 버튼을 활성화한다. | 설문 상세 조회 |
| | 응답 버튼을 눌러 응답 화면으로 이동한 후, 각 문항에 응답을 입력하고 완료 버튼을 눌러 제출할 수 있다. | 설문 응답 |
